🔒 Provably Fair System

Every pack opening uses a cryptographic random number generator seeded with a combination of server seed, client seed, and a nonce (counter). This ensures that outcomes cannot be manipulated after the fact.

How it works:

1. Before you open a pack, a server seed hash is generated and shown to you.
2. You can provide your own client seed, or use the auto-generated one.
3. The outcome is determined by: SHA-256(serverSeed + clientSeed + nonce)
4. After the round, the server seed is revealed so you can verify the hash matches.
